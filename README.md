# RandomPassGen
A secure web-based random password generator built with HTML5, CSS3, and JavaScript, featuring customizable length, character type selection, and one-click copy functionality.
## Key Features
- **Customizable Complexity:** Options to include uppercase letters, lowercase letters, numbers, and special symbols.
- **Adjustable Length:** Users can specify the exact length of the password for different security requirements.
- **One-Click Copy:** Integrated clipboard functionality for immediate use.
- **Security Assessment:** Real-time visual feedback on password strength (Weak, Medium, Strong).
- **Responsive Interface:** Modern, clean design using CSS Flexbox/Grid for all device types.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3
- **Logic:** Vanilla JavaScript (ES6+)
- **Security:** Built-in algorithms for true character randomness.

## What I Learned
- **Cryptographic Randomness:** Implementing math-based logic to ensure unpredictability and resist dictionary attacks.
- **Input Validation:** Managing state to ensure at least one character type is selected before generation.
- **Clipboard API:** Using modern browser APIs to provide a seamless "copy to clipboard" experience.
